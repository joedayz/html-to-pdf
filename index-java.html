Temario Completo (Actualizado 2024 – Java SE 21)

1. Introducción a Java – Explicar características Java y Conceptos orientados a objetos; Definir, compilar y ejecutar un programa Java
2. Tipos primitivos, Operadores y Control de Flujo – Declarar, inicializar primitivos y entender restricciones; Describir operadores Java, Asignación, operadores aritméticos y type casting; Representación de numeros binarios y operadores lógicos bitwise; Operadores de igualdad, relacionales, condicionales, ternario y if/else; Control de flujo usando Switch
3. Texto, Fechas, Tiempo y Objetos Numéricos – Inicialización de String y Operaciones; Indexing String, Objetos de texto mutables y Blocks; Describir Wrapper de primitivos, Clases BigDecimal y encadenamiento de métodos; Manejar fechas y tiempo; Representar lenguajes y paises, formatear y parsear números, fechas y tiempo; Describir localización y formateo de clases
4. Clases y Objetos – Ejercicio de creación de clases; Documentar clases
5. Diseño de clases – Sobrecarga de métodos; Constructores; Encapsulación e inmutabilidad usando modificadores de acceso; Explicando el provisionamiento de memoria en Java y pase de parámetros; Explicando la recolección de memoria en Java
6. Herencia y uso de Records – Herencia de clases y reutilización de código usando herencia; Instanciar clases y acceso a objetos, reglas de casting de tipos de referencia; Verificando el tipo de un bojeto usando el instanceof, ejecutar type casting de tipos de referencia; Referenciar codigo dentro de objetos y definir constructores de subclases; Sobreescritura de métodos y uso de polimorfismo; Definir clases abstractas, final, sealed e interfaces; Sobreescribir operaciones de la clase Object; Definir clases Record, Pattern matching para switch; Definir y usar métodos Factory
7. Interfaces y Genéricos – Describir interfaces; Implementar interfaces; Heredar de interfaces y usar interfaces funcionales; Usar genéricos; Explorar interfaces comunes de Java y usar el patrón composition
8. Arrays y Loops – Declarar, crear e inicializar arrays; Usar arrays multidimensionales y clases Array; Procesar arrays en un loop; Usar loops embebidos, break y continue
9. Colecciones – Introducir el Java Collection API, interfaces y clases de implementación; Crear y manejar List, Set, Deque, HashMap y contenidos; Iterar, a través de, colecciones, usar Sequenced Collections y clases Collections; Acceder a colecciones concurrentemente y prevenir corrupción de colecciones
10. Clases Nested y Expresiones Lambdas – Introducir clases Nested e Inner; Clases Inner anónmias e interfaces funcionales; Definir parámetros de una expresión lambda y body; Usar referencias a métodos y métodos no abstractos en interfaces
11. API Java Stream – Describir el API Java Stream y procesar stream pipelines; Implementar interfaces funcionales y Primitive y variantes Bi-argument; Procesar Stream Pipelines y ejecutar operaciones Stream; Procesar Stream y agregar stream data; Usar collectors, ejecutar grouping y partitioning del contenido de un Stream; Describir el procesamiento de un stream paralelo, guía y restricciones; Usar Spliterator
12. Manejo de Excepciones, Logging y Debugging – Java Logging; Manejo de excepciones; Debugging
13. API Java IO – Explorar el API Java IO; Leer y escribir data binaria y caracteres; Usar entrada estándar, salida y clase Console; Entender la serialización, el grafo de objetos serializable y serialización de objetos; Personalizar la serialización y deserialización; Construyendo el Filesystem Paths y navegando en el Filesystem; Operar en objetos Path y Properties, manejar archivos temporales y directorios; Explorar ejemplos de I/O: trabajar con archivos Zip y acceder a recursos HTTP
14. Concurrencia en Java y Multithreading – Conceptos e implementación de Hilos; Manejar el ciclo de vida de un Hilo, interrumpir y bloquear un Hilo; Wait y propiedades; Manejar e implementar un Executor Service; Discutir problemas de locking y asegurar ejecución segura de hilos; Acciones atomicas no bloqueantes, lock intrinseco y automatización de concurrencia; CPU vs IO Bound Concurrent Tasks y uso de Virtual Threads
15. Módulos y Despliegue – Compilar, empaquetar y ejecutar aplicaciones no modulares; Describir el Java Module System; Crear dependencias de módulos y exportar contenido de módulo; Definir y usar Module Services; Compilar, empaquetar, ejecutar y migrar aplicaciones modulares; Crear y ejecutar una imagen runtime personalizada
16. Anotaciones
17. Conectividad a Base de Datos con JDBC
18. Seguridad Java
19. Genericos avanzado – Erasure y uso de genéricos
20. Aplicaciones Java en Oracle Cloud – Principios de diseño de aplicaciones cloud
21. Temas adicionales – Describir patrones de diseño, Java IO, File Watch Service
